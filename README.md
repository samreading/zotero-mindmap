#### MindMap Outline short cuts

| Feature                               | Short Cut                                 |
| ------------------------------------- | ----------------------------------------- |
| New Child Node                        | Tab and Enter                             |
| Zoom in/out                           | CTRL + Mouse wheel                        |
| Delete node                           | Delete/Backspace                          |
| Edit node                             | Space/Double click node                   |
| Undo                                  | CTRL/CMD + Z                              |
| Redo                                  | CTRL/CMD + Y                              |
| Delete summary/boundary/related link  | Delete/Backspace                          |
